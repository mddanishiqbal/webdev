Responsive Image Gallery

Lightbox Image Gallery Web Applicataion using HTML, CSS, JavaScript & Lightbox (JavaScript library). In this App, When you will click on any image of the galley then the same image will popup in the center of website and you can also check the next and previous images by clicking on next and previous icons. All image will have captions that will be displayed just below the image.

In this project, we use :

1 - React functional components and their reusability

2 - Redux Toolkit to manage the state of our application

3 - Using Tailwind and make the app responsive on all devices

4 - To fetch data from unlimited sources using RapidAPI
